    Function WebSocket.sendMessage

    Send a custom message. isFIN is true by default and should be true for most cases. You want to set it to false if you are sending a message in parts. The last part should have isFIN set to true. The masked parameter is false by default and should be false for most cases. It is used if a message is sent from a client to a server.
